# 🧬 AI-Driven IVF Sperm Morphology Classification System

## 📌 Overview
This project is an AI-powered decision support system that predicts sperm morphology types using machine learning.

## 🚀 Features
- Random Forest classification model
- Streamlit interactive web app
- SQLite database for prediction history
- Power BI dashboard for analytics
- Batch prediction support
- Feature importance visualization

## 🧠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- SQLite
- Power BI

## 📊 Dataset
HuSHeM Sperm Morphology Dataset
